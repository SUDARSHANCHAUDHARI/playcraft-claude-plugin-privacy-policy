Privacy Policy

Last updated: April 6, 2026

This Privacy Policy applies to the PlayCraft Claude Code plugin, developed by SUDARSHANCHAUDHARI / SudarshanTechLabs.

What PlayCraft Does

PlayCraft is a Claude Code plugin that helps Android developers automate Google Play Store publishing tasks — store listings, ASO audits, release notes, policy declarations, and screenshot copy — entirely on your local machine.

Data Collection

PlayCraft does not collect, transmit, or store any user data. Specifically:

  • All listing generation and auditing happens locally in your Claude Code session
  • No data is sent to any external server by this plugin
  • Your developer details (name, company, email, GitHub username, country) are stored locally in a .claude-plugin-config.json file in your project directory only
  • The setup command automatically adds this config file to .gitignore to prevent accidental commits
  • No analytics, crash reporting, or telemetry is included in this plugin

Third-Party Services

PlayCraft itself uses no third-party services. It runs entirely inside Claude Code (by Anthropic). Your inputs and outputs are processed by Claude according to Anthropic's Privacy Policy. No data passes through any PlayCraft servers — there are no PlayCraft servers.

Google Play Console

PlayCraft does not connect to Google Play Console on your behalf. It generates copy and checklists that you then apply manually in Play Console. No Google credentials are ever accessed or stored by this plugin.
