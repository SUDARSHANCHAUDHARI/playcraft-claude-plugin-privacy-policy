Privacy Policy — PlayCraft

Last updated: April 6, 2026  ·  Developer: SudarshanTechLabs

Overview

PlayCraft is a Claude Code plugin that helps Android developers automate Google Play Store publishing tasks — store listings, ASO audits, release notes, policy declarations, and screenshot copy.

This privacy policy explains what information PlayCraft touches and how it is handled. The short version: PlayCraft does not collect, store, or transmit any personal data to any server.

What We Don't Collect

PlayCraft does not collect:

  • Names, email addresses, or contact information
  • App usage data or analytics
  • Device identifiers or crash reports
  • Google Play Console credentials or API keys
  • Any data from your app's users

How the Plugin Config Works

When you run /playcraft:setup, the plugin saves your developer details (name, company, email, GitHub username, country, and locale preferences) to a file called .claude-plugin-config.json in your project directory on your local machine.

This file is stored only on your device. It is never uploaded to any server by PlayCraft. The setup command automatically adds this file to your project's .gitignore so it is not accidentally committed to version control.

Permissions and Access

PlayCraft is a text-based plugin. It:

  • Reads and writes local files on your machine (to create config and policy output files)
  • Does not access the internet independently
  • Does not connect to any PlayCraft servers — there are no PlayCraft servers
  • Does not access Google Play Console on your behalf

All output (store listings, release notes, audit reports, declaration checklists) is generated locally by Claude and displayed in your Claude Code session or saved to files you specify.

Third-Party Services

PlayCraft runs inside Claude Code, developed by Anthropic. When you use PlayCraft, your inputs and outputs are processed by Claude (Anthropic's AI model) according to Anthropic's Privacy Policy.

PlayCraft itself does not integrate with, send data to, or receive data from any third-party service.

Data Storage

No data is stored by PlayCraft on any remote server. The only storage that occurs is the .claude-plugin-config.json file written to your local machine during setup, and any output files you choose to save locally. You can delete these at any time.

Children's Privacy

PlayCraft is a developer tool intended for adults. It is not directed at children under 13, and no information is knowingly collected from children.

Changes to This Policy

If this privacy policy changes, the updated version will be posted at this URL with a new "Last updated" date. Changes take effect when posted.
